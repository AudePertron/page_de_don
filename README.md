# page_de_don
 projet individuel<br>
La base de données est hébergée sur MongoDB/Atlas, elle contient 
  - la collection "donateurs": composée de documents possédant les champs: Nom, prénom, adresse, email et somme promise
  - la collection "membres": qui liste le nom d'utilisateur et le mot de passe des personnes ayant l'autorisation de voir la page donateurs.
